# Java Servlets

This module focuses on Servlet-based web application development using Java.

## Technologies
- Java
- Servlet API
- Apache Tomcat
- HTML

## Topics Covered
- Servlet Life Cycle
- doGet() and doPost()
- Form Handling
- Request & Response
- Session Management
- Login & Authentication

## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat
3. Run on server

## Learning Outcome
- Handling client requests
- Server-side programming using Java
- Web application flow understanding
