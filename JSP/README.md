# JavaServer Pages (JSP)

This module contains JSP programs for creating dynamic web pages.

## Technologies
- JSP
- HTML
- CSS
- Apache Tomcat

## Topics Covered
- JSP Life Cycle
- JSP Directives
- Scriptlet, Expression, Declaration
- Implicit Objects
- JSP with Servlet Integration

## How to Run
1. Deploy project on Apache Tomcat
2. Access JSP pages via browser

## Learning Outcome
- Dynamic page creation
- Integration of JSP with backend logic
- Clean separation of view layer
