# JDBC (Java Database Connectivity)

This module contains programs related to JDBC for connecting Java applications with a relational database.

## Technologies
- Java
- JDBC API
- MySQL
- Eclipse IDE

## Topics Covered
- JDBC Architecture
- DriverManager & Connection
- Statement & PreparedStatement
- ResultSet
- CRUD Operations
- Exception Handling

## How to Run
1. Create database in MySQL
2. Update database URL, username & password
3. Run Java class

## Learning Outcome
- Database connectivity using Java
- Writing SQL queries in Java
- Handling database operations
